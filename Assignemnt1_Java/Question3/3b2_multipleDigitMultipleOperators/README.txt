This folder contains answer of part b of ques3 with multiple operator input and multi digit operands.


For example: 32+24-11
The above input will return 45

Press enter when 3 is highlighted
Press enter when 2 is highlighted

Press space when + is highlighted

Press enter when 2 is highlighted
Press enter when 4 is highlighted

Press space when - is highlighted

Press enter when 1 is highlighted
Press enter when 1 is highlighted

Press enter when evaluate is highlighted

After this answer will be displayed on the screen.

Press Space on clear after the above process to clear the screen or you can continue with first operand as ans of previous query. Here you can add as much operators and operands with multiple digits in a single expression itself.

Note: To compute the expression, * and / are given higher prioirty than + and - and between same prioirty operators, left to right evaluation is done.


To compile the code, type:
javac -d . *.java

To run the code after compiling, type:
java task3b2.Calculator

============================================

Note: 
You can also compile using the following command:
make

To run the code after compiling, type:
java task3b2.Calculator

To remove task3b2 folder which is generated after compiling type:
make clean
