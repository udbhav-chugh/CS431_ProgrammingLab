This folder contains the code for single digit input with single operand in between:

For example: 3+2
The above input will return 5

Press enter when 3 is highlighted
Press enter when + is highlighted
Press enter when 2 is highlighted

After this answer will be displayed on the screen.

Press Enter on clear after the above process to clear screen or on any other operator to continue with the above answer as first operand.


To compile the code, type:
javac -d . *.java

To run the code after compiling, type:
java task3a1.Calculator

============================================

Note: 
You can also compile using the following command:
make

To run the code after compiling, type:
java task3a1.Calculator

To remove task3a1 folder which is generated after compiling type:
make clean
